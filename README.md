# Antigravity ポートフォリオプロジェクト

このプロジェクトは、モダンなWeb技術を使用して構築された個人ポートフォリオWebサイトです。

## 技術スタック (Tech Stack)

### コア技術 (Core)
- **[React](https://react.dev/)**: ユーザーインターフェースを構築するためのJavaScriptライブラリ。
- **[TypeScript](https://www.typescriptlang.org/)**: JavaScriptを拡張した静的型付け言語。
- **[Vite](https://vitejs.dev/)**: 次世代のフロントエンドツール。

### スタイリング & UI
- **[Tailwind CSS](https://tailwindcss.com/)**: ユーティリティファーストのCSSフレームワーク。
- **[PostCSS](https://postcss.org/)**: JavaScriptでCSSを変換するためのツール。
- **[Lucide React](https://lucide.dev/)**: 統一された美しいアイコンライブラリ。
- **[Framer Motion](https://www.framer.com/motion/)**: React用の本番対応モーションライブラリ。

### 開発ユーティリティ
- **clsx** & **tailwind-merge**: 条件付きでクラス名を構築・結合するためのユーティリティ。

## はじめに (Getting Started)

### 前提条件
- Node.js (LTSバージョン推奨)
- npm

### インストール方法

1. リポジトリをクローンします。
2. 依存関係をインストールします:
   ```bash
   npm install
   ```

### 開発

開発サーバーを起動します:
```bash
npm run dev
```

### ビルド

本番環境用にビルドします:
```bash
npm run build
```
